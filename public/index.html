Notizen API

AI-powered text summarization service

Overview

Notizen API provides text summarization capabilities powered by Claude AI. Send text content and receive intelligent summaries tailored to your needs.

Endpoints

POST /api/summarize

Summarize text content using Claude AI.

Request Format

Send a POST request with your Anthropic API message format:

{
  "model": "claude-3-5-sonnet-20241022",
  "max_tokens": 1024,
  "messages": [
    {
      "role": "user",
      "content": "Your text to summarize here"
    }
  ]
}

Headers

  • Content-Type: application/json

Example Usage

fetch('https://your-domain.vercel.app/api/summarize', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    model: 'claude-3-5-sonnet-20241022',
    max_tokens: 1024,
    messages: [{
      role: 'user',
      content: 'Summarize this text: ...'
    }]
  })
})
.then(res => res.json())
.then(data => console.log(data));
Note: The API requires a valid Anthropic API key to be configured in the environment variables (ANTHROPIC_API_KEY).

CORS Support

This API supports Cross-Origin Resource Sharing (CORS) and can be called from any domain.

Response

The API returns the response directly from Claude AI, including the generated summary and token usage information.
